A modern weather tracking web application built for monitoring weather conditions in Simferopol, Ukraine. Available as a Telegram Mini App with multi-language support.
 ✨ Features
- Real-time current weather conditions
- 48-hour hourly forecast with charts
- 7-day weather forecast
- Historical weather data (7-30 days)
- Smart weather alerts (extreme temps, high wind, precipitation)
- Multi-language: English & Russian 🇬🇧 🇷🇺
- 3 themes: Light, Dark, Blue (calm/spring mood)
- Telegram Mini App integration
- Mobile-responsive design
 🛠️ Tech Stack
- React 19 + Vite
- Open-Meteo API (free, no key required)
- react-i18next (translations)
- Recharts (charts)
- Telegram WebApp SDK
 🚀 Live Demo
**Web:** https://dyestuff.github.io/simferopol-weather
**Telegram:** Search for @weather_simferopol_bot
 📱 How to Run Locally
git clone https://github.com/dyestuff/simferopol-weather.git
cd simferopol-weather
npm install
npm run dev
🌐 API
Uses Open-Meteo (https://open-meteo.com/) - free weather API, no registration required.
